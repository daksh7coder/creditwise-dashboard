
# CreditWise Dashboard 💳

CreditWise Dashboard is a React-based web application that helps users manage multiple credit cards, track spending, and stay on top of bill payments through a clean and interactive interface.

This project is built as part of a React Capstone Project and focuses on real-world usability, state management, and data visualization.

---

## 🚀 Features

* 💳 Add and manage multiple credit cards
* 📊 Interactive dashboard with total spend and credit utilization
* ⏰ Bill due date tracking and reminders
* 📈 Spending analytics using charts
* 📜 Transaction history overview
* 🔔 Alerts for upcoming payments
* 📱 Fully responsive UI (mobile + desktop)

---

## 🧠 Project Objective

The goal of this project is to build a practical and scalable frontend application using React that simulates a real-world financial management tool.

---

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Routing:** React Router
* **State Management:** Context API / useState
* **Styling:** Tailwind CSS / CSS Modules
* **Charts & Visualization:** Recharts / Chart.js

## ⚙️ Installation & Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/creditwise-dashboard.git
   ```

2. Navigate to the project directory:

   ```bash
   cd creditwise-dashboard
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Run the development server:

   ```bash
   npm start
   ```
